# Introduction

DROPIFY® documentation lives here. Use the sidebar to explore topics.

## What is DROPIFY®?

Dropify helps you build hype for your Shopify merchandise drops with unlockable timers, previews, and more.

## Quick start

- Install the app
- Configure your drop
- Publish and share
